## Asset Requirements Checklist

After downloading, rename/organize files as follows:

```
sprites/
├── character/
│   ├── dev-idle.png      # Character idle animation frames
│   ├── dev-typing.png    # Character typing animation
│   ├── dev-thinking.png  # Character thinking pose
│   └── dev-walk.png      # Character walking animation
├── furniture/
│   ├── desk.png          # Office desk
│   ├── chair.png         # Office chair
│   ├── monitor.png       # Computer monitor
│   └── keyboard.png      # Keyboard
├── background/
│   ├── whiteboard.png    # Whiteboard on wall
│   ├── corkboard.png     # Corkboard with notes
│   └── server-rack.png   # Server rack
└── ui/
    ├── status-dot.png    # Status indicators
    └── buttons.png       # UI buttons
```

## Creating Custom Sprites

If you can't find suitable free assets, create custom sprites using:

1. **Piskel** (free, web-based): https://www.piskelapp.com/
2. **Aseprite** ($20, professional): https://www.aseprite.org/
3. **Pixelorama** (free, open source): https://orama-interactive.itch.io/pixelorama

### Sprite Size Guide

For the current 180x320 internal resolution:
- Characters: 16x24 to 32x32 pixels
- Furniture: 16x16 to 32x32 tiles
- UI elements: 8x8 to 16x16 for buttons/indicators
